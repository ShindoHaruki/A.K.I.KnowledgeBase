# SF6 A.K.I. Knowledge Base

## Goal
Street Fighter 6 の A.K.I. の通常技後ケースを管理する。

## Rules
- `data/aki_normals.yml` を正とする
- PlantUMLは `data/aki_normals.yml` から更新する
- 不明なフレーム値は `TODO` のまま残す
- 更新時は `notes/changelog.md` に変更内容を書く
- 推測でフレーム値を埋めない
- 日本語でコメント・分岐を書く

## Files
- Main data: `data/aki_normals.yml`
- Rules: `data/rules.yml`
- PlantUML: `plantuml/aki_normal_cases.puml`
- Changelog: `notes/changelog.md`
